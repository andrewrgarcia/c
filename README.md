#### args.c
A simple c code example which shows you how to pass inputs as arguments in terminal with a c program. 
compiling:
`gcc args.c -o args`
deployment:
`./args [first_word] [second_word] [. . .] [infinite-1_word]`

make a conditional statements AND use args input.
